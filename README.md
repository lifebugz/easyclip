# EasyClip

A simple, cross-platform desktop media trimmer for non-technical users.
Trim video and audio files without learning complex software — drag the side anchors to trim ends, click the timeline to remove middle sections. All cuts are lossless via FFmpeg stream-copy / concat-demuxer; zero re-encoding.

EN + HE with full RTL layout. MIT-licensed app, LGPL FFmpeg as bundled sidecar.

See `docs/superpowers/specs/2026-04-26-easyclip-v1-design.md` for the full design.
